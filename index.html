  • Atalanta Goal Alert

    App die waarschuwt als Atalanta BC scoort

    De app waarvan je nog niet wist dat je hem nodig had.

    Gemaakt in VueJs, met een API call naar football-data.org.

    Progressive Web App (PWA) proof door middel van een serviceworker die je netjes op tijd notificaties stuurt als er iets aan de hand is.

    VueJs · Semantic HTML · PWA · Figma · NodeJs

    Naast de notificaties bouwen heb ik ook:

    • Een node server gebouwd en gehost via Heroku
    • Ervoor gezorgd dat de server continu voetbaldata binnenhaalt
    • Gebruik gemaakt van flex-direction: row-reverse; voor dit trucje
